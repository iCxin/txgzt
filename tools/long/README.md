# 微信接龙对比工具

这是一个简单的网页工具，用于比较微信接龙数据和已知名单，快速找出未接龙的人员。

## 功能特点

- 自动识别微信接龙中的名字（忽略前面的序号和班级信息）
- 快速对比找出未接龙人员
- 简洁明了的界面设计
- 支持移动设备访问

## 使用方法

1. 在左侧文本框中粘贴微信接龙数据，格式如下：
   ```
   #接龙
   已加入入团积极分子培训群
   
   1. 23水利3班张三
   2. 24土木1赵六
   ...
   ```

2. 在右侧文本框中粘贴完整名单，每行一个名字，例如：
   ```
   张三
   赵六
   钱一
   ...
   ```

3. 点击"开始对比"按钮，系统将自动分析并显示未接龙的人员名单。

4. 如需清空数据重新输入，可点击"清空数据"按钮。

## 小技巧

- 双击空白的文本框可以填充示例数据，方便测试
- 接龙数据中的名字会自动从包含班级信息的文本中识别出来
- 支持不同格式的序号，如"1."、"1、"、"1 "等
- 支持各种格式的接龙数据，包含班级、年级等信息也能正确识别名字

## 本地运行

1. 下载所有文件到本地文件夹
2. 直接在浏览器中打开`index.html`文件即可使用
